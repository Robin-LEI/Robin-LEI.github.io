环境

  1. npm i -g @angular/cli
  2. ng -v
  3. ng new project
  4. ng serve –open
  5. ie8不支持angular
  6. ng new project –skip-install // 不会安装依赖

创建组件

  1. ng g component 名称
  2. ng g component components/new // 在app目录下，创建一个文件夹，文件夹下面创建一个new文件夹，里面时new的组件内容
  3. 之前传统网站，我们想把头部和底部等拆成单个文件来写，但是受困于当时技术，不容易实现，angular的组件帮助我们实现了这一梦想

属性修饰符

  1. public：在当前类、子类、类的外部类也可以访问
  2. private：当前类
  3. protected：当前类及其子类

服务

  1. 两个组件是不能进行通信的，A组件不能调用B组件的方法
  2. 如果我们要定义一个公共的方法，使得在A、B组件都能使用，我们需要把这个方法放在服务当中
  3. 创建服务

生命周期函数

  1. ngOnInit()页面刷新会触发这个生命周期函数,组件和指令初始化完成，并不是dom加载完成
  2. ngAfterViewInit() // 视图加载完成以后触发，可以获取dom

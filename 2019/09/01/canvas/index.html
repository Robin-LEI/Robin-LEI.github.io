canvas

2019-09-01

简介

  1. HTML5新标签
  2. 本身不具备绘画功能
  3. 是一个矩形区域的画布
  4. 拥有多种绘制图形图像的方法

应用领域

  1. 游戏
  2. 广告
  3. 可视化数据

设置宽高

  1. 设置宽度和高度，通过属性设置，不要通过css设置
  2. 对于一个canvas标签来说，使用的时候一定要设置width和height属性，否则绘制的图形有问题

兼容性

  1. IE9才支持
  2. 不支持会认为是div
  3. 兼容性写法

步骤

  1. 拿到canvas标签
  2. 拿到上下文
  3. 绘制

常用方法

  1. moveTo // 移动画笔到指定位置，不设置，没有位置
  2. lineTo // 画线
  3. stroke // 描边
  4. closePath // 路径闭合
  5. fill // 填充

常用属性

  1. strokeStyle // 设置描边颜色
  2. lineWidth // 线宽
  3. fillStyle // 设置填充颜色
